# Checklist de Producción CapCut — HUMANOS

> Usa esta lista para cada video. El objetivo es producir rápido con calidad consistente.
> Una historia por archivo. Una historia por sesión.

---

## PRE-PRODUCCIÓN

- [ ] **Seleccionar personaje** desde Supabase (estado: `gabo_listo`)
- [ ] **Leer guion completo** antes de grabar. Entenderlo, no memorizarlo.
- [ ] **Elegir versión del guion:** 30s / 45-75s / 90s
- [ ] **Imprimir o tener el guion visible** en pantalla durante grabación

---

## GRABACIÓN DE VOZ

- [ ] **Espacio silencioso.** Sin ruido de fondo, ventiladores, notificaciones.
- [ ] **Micrófono** a distancia correcta (15-20 cm aprox)
- [ ] **Hacer 2-3 takes completos** del guion antes de editar
- [ ] **Grabar secciones por separado** si alguna parte falla repetidamente
- [ ] **Revisar audio** antes de pasar a edición (escuchar completo una vez)

---

## EDICIÓN EN CAPCUT

### Estructura visual (seguir orden de bloques)

- [ ] **Bloque 1 — Hook (0-3s):** Texto grande + visual fuerte. Sin voz todavía si aplica.
- [ ] **Bloque 2 — Tarjeta de personaje (1-2s):**
  ```
  HUMANOS
  [NOMBRE]
  [Frase descriptor]
  ```
- [ ] **Bloque 3 — Origen:** Foto del personaje / contexto / infancia / país
- [ ] **Bloque 4 — Conflicto:** Visual oscuro, palabra de impacto
- [ ] **Bloque 5 — Giro:** Cambio musical o pausa visual. Texto: "Pero entonces..." / "Y ahí todo cambió."
- [ ] **Bloque 6 — Resultado:** Visual de logro / empresa / impacto
- [ ] **Bloque 7 — Cierre:** Pantalla limpia. Firma de Jota. HUMANOS.

---

### Subtítulos

- [ ] **Activar subtítulos automáticos** en CapCut (Auto Captions)
- [ ] **Revisar y corregir** errores de transcripción (especialmente nombres propios)
- [ ] **Ajustar tamaño:** grande, sobrio. No karaoke.
- [ ] **Resaltar palabras clave** manualmente:
  - NO / FRACASÓ / APPLE / MILLONES / CUPONES / VIDRIO / INMIGRANTE / etc.
- [ ] **Máximo 1-2 líneas** de subtítulo en pantalla a la vez

---

### Música

- [ ] **Seleccionar música documental** (no motivacional, no TikTok genérico)
  - Tono: tensión ligera / misterio / descubrimiento / emoción contenida
- [ ] **Ajustar volumen:** música baja durante narración (~15-20%), sube en pausas y cierre
- [ ] **Silencio en [PAUSA] marcadas** — dejar respirar la narración

---

### Visual y tipografía

- [ ] **Fondo oscuro** (negro / gris carbón)
- [ ] **Texto blanco** para cuerpo
- [ ] **Color acento** en palabras clave (amarillo / azul eléctrico / rojo profundo)
- [ ] **Fuentes:** Bebas Neue / Anton / Montserrat ExtraBold para títulos
- [ ] **Zoom lento / paneo suave** en fotos estáticas
- [ ] **Sin transiciones infantiles.** Cortes limpios o fundidos sutiles.

---

## EXPORTACIÓN

- [ ] **Formato:** 9:16 (vertical)
- [ ] **Resolución:** 1080x1920 mínimo
- [ ] **Calidad:** Alta / máxima disponible
- [ ] **Nombre del archivo:**
  ```
  HUMANOS_[NombrePersonaje]_v[número].mp4
  ```
  Ejemplo: `HUMANOS_JanKoum_v1.mp4`

---

## PUBLICACIÓN

- [ ] **Instagram Reels:** Subir con copy de descripción + hashtags
- [ ] **TikTok:** Subir (puede requerir ajuste de hashtags)
- [ ] **YouTube Shorts:** Subir con título y descripción adaptada
- [ ] **Comentario fijado:** Pegar texto del Archivo Humano con link en bio
- [ ] **Verificar subtítulos en plataforma** (a veces se desincroniza)

---

## POST-PUBLICACIÓN

- [ ] **Registrar URLs** de cada plataforma en Supabase
- [ ] **Cambiar estado** en Supabase a `publicado`
- [ ] **A las 48 horas:** revisar métricas iniciales
- [ ] **A los 7 días:** registrar métricas finales en Supabase
  - Vistas / Retención % / Comentarios / Compartidos / Guardados
- [ ] **Decisión:** ¿Merece serie? Actualizar campo `merece_serie` en Supabase

---

## Regla de producción

> Publica imperfecto antes de no publicar perfecto.
> El mayor riesgo no es publicar algo que no está al 100%.
> El mayor riesgo es enamorarse tanto de la visión que nunca publicas.
